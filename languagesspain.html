How it was calculated?

Map

If you visit this repository, you could find data about developers from different parts in Spain. I took this data and made a count of the languages.

Graph

In this repository, there are data about Spain. You could took this data and calculate percentage of use of this languages.

How to insert in my website?

You can donwload the script from here (map) and here (graph). You will need add to your web:

  • jQuery
  • Char.JS
  • RaphaelJS
  • Map generator*
  • A table whose ID will be "legend"
  • A canvas whose ID will be "char"
  • A table whose ID will be "contributions"

If you want to create your own map, you could see this repo

It is not obligatory, but I would appreciate if you add a link in your page to this web
